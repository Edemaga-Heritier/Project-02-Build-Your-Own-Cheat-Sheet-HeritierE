HTML Table Reference

Table Tags

Tag name Description
<audio> Audio Pour lire un fichier audio.
<aside> Aside Cette balise permet de regrouper des informations non essentielles relatives au site Web.
<br> Br Utilisée dans un paragraphe, cette balise permet de créer un saut de ligne.
<p> P Définis un paragraphe contenant une ou plusieurs phrases. Cette balise ne doit pas être utilisée si une autre balise est mieux indiquée.
<button> Button Pour un bouton. il peut être utilisé comme commande.

Attribute Name Description

<href> href Définit l’adresse du lien
